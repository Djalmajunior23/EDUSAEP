# 🚀 PROMPT MESTRE — EDUAI CORE ULTRA (PLATAFORMA DE INTELIGÊNCIA PEDAGÓGICA DE ALTA PERFORMANCE)

**Status:** ATIVO
**Revisão:** 1.0
**Foco:** Escala, Inteligência Pedagógica Avançada e Protagonismo Docente

Este documento é a estrela-guia (North Star) do desenvolvimento da plataforma EduAI Core. Todas as próximas funcionalidades, integrações e refatorações DEVEM estar alinhadas a estas 80 epics e aos pilares de arquitetura aqui definidos.

## 🎯 OBJETIVO GLOBAL
Construir um sistema que ajude o professor a decidir melhor, reduza drasticamente o trabalho manual, automatize processos pedagógicos, analise profundamente o aprendizado, identifique problemas antes que aconteçam, aumente a qualidade do ensino e escale para múltiplas instituições com segurança e performance.

## ⚠️ DIRETRIZES CRÍTICAS
1. Manter simplicidade de uso.
2. Manter foco pedagógico.
3. Preservar protagonismo do professor.
4. Garantir alta performance.
5. Permitir explicabilidade.
6. Ser modular e escalável.
7. Permitir evolução contínua sem retrabalho.

*(Para a lista completa detalhada, consulte AGENTS.md e o mapa de blocos oficiais do sistema)*

## 🏗️ ARQUITETURA OFICIAL
1. **Firebase**: Camada de dados centralizada (Auth, Firestore, Storage).
2. **Motor Pedagógico**: Classes utilitárias e regras TypeScript no Node/React.
3. **IA (Gemini)**: Geração de conteúdo, análise, recomendação, copiloto e explicabilidade.
4. **n8n**: Automações externas, alertas, integrações webhook, CRON jobs.
5. **Frontend**: React.js com Tailwind CSS (Vite), priorizando lazy loading e state management limpo.

## 🚀 ROTEIRO DE EXECUÇÃO (ROADMAP)
- **Fase 1**: Dashboards, Observatórios, Mapa de Erros.
- **Fase 2**: Automação, Geração, Correção Assistida.
- **Fase 3**: Inteligência, Predição, Agrupamento Preditivo.
- **Fase 4**: Avaliação Avançada (Imagens, Código, Simuladores).
- **Fase 5**: Inteligência Artificial Autônoma Completa.

**LEMA:** Não simplifique. Não resuma. Entregue solução robusta, escalável e pronta para produção.
